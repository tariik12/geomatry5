DOM means : Document Object Model (DOM)
which is a programming interface for web documents . is an (API) application programming interface for HTML and XML documents. purpose of DOM: scripts to dynamically access and update the content, structure, and style of a document which provides a tree-like structure of document. It's create, change, or remove elements from the document and node represents a part of the document, such as an element, attribute, or text.

Using DOM :
There are 5 ways in which we are can select elements in a DOM using selectors.
1. getElementsByTagName()
Example: const element = document.getElementById('my-id');
2.getElementsByClassName()
Example: const elements = document.getElementsByClassName('my-class');;
3.getElementById()
Example: const elements = document.getElementsByTagName('div');
4.querySelector()
Example: const element = document.querySelector('.my-class');
5.querySelectorAll()
Example: const elements = document.querySelectorAll('.my-class');
